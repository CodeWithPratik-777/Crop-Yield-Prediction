🌾 Crop Yield Prediction

A data-driven web application that predicts crop yield based on environmental and soil parameters such as weather, rainfall, and soil conditions. This project aims to support farmers and stakeholders in making informed decisions to improve agricultural productivity and promote sustainable farming.

🚀 Features
📊 Predict crop yield using machine learning models
🌦️ Considers weather and rainfall data
🌱 Includes soil parameter analysis
💻 User-friendly interface for farmers
📈 Data visualization for better insights

🛠️ Tech Stack

Frontend:
Streamlit
HTML5, CSS3, JavaScript

Machine Learning:

Python
Scikit-learn / Pandas / NumPy
